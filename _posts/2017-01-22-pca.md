Including Plots
---------------

Plot first and second principal components.

``` r
plot(my_pca$x[,1], my_pca$x[,2], col = iris$Species, xlab = "PC1", ylab = "PC2", pch=19)
legend('topright', legend = levels(iris$Species), col = 1:3, pch = rep(19,3))
```
